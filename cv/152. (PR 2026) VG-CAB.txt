a b s t r a c t
Visible-Infrared object detection aims to utilize the complementarity of diﬀerent modalities to improve the accuracy of object classiﬁcation and localization in complex environments. However, most existing methods prioritize detection performance while neglecting network complexity, limiting their real-world applications. To
this end, we propose a lightweight modal-guided cross-attention fusion network (LCAFNet) for visible-infrared
object detection, which is composed of a visible-guided cross-attention block (VG-CAB), an infrared-guided crossattention block (IG-CAB), and a gated fusion block (GFB). The VG-CAB and IG-CAB utilize attention weights from
one modality to guide information aggregation from another modality, enabling cross-modal information interaction and feature fusion from two diﬀerent perspectives. These two blocks generate complementary features
containing visible (VIS) and infrared (IR) information, resulting in a comprehensive and robust multimodal feature representation. Beneﬁting from the enhanced and complementary features generated by the VG-CAB and
IG-CAB, the GFB achieves an adaptive and complete fusion between these features through a gating strategy.
Further, shallow VIS and IR features extracted from a dual-branch backbone network are applied to mine and
integrate complementary features, which have more spatial and edge information, thus improving the localization and classiﬁcation capabilities of the detection model. Extensive experiments demonstrate that our proposed
LCAFNet obtains better detection performance and lower network complexity compared with other excellent
models on ﬁve commonly used public datasets. Speciﬁcally, on the DroneVehicle dataset, our LCAFNet outperforms the state-of-the-art model by 1.6% mAP50, but has only one-eighth the number of network parameters.
The source code for our LCAFNet is accessible at https://github.com/WenCongWu/LCAFNet.

1. Introduction
Object detection is an indispensable computer vision task, which
is widely used in traﬃc management, autonomous driving, and scene
surveillance [1,2]. However, real-world scenarios present several challenges that aﬀect detection performance, such as object occlusion, small
object size, low illumination, etc. Monomodal detection methods [3–5]
tend to face problems such as limited robustness, low detection accuracy, and poor localization ability in complex environments. Multimodal
(VIS and IR) methods [6–8] suﬀer from high network complexity, insufﬁcient information utilization, and intermodal discrepancy. These challenges inspired us to develop a lightweight and eﬀective VIS-IR object
detection model.

Recently, VIS-IR object detection approaches are mainly divided
into two schemes: CNN-based [9–11] and transformer-based methods [12–14]. CNN-based detection models, limited by their local
receptive ﬁelds, struggle to eﬀectively capture cross-modal global
semantic relationships and long-range dependencies, resulting in
insuﬃcient feature interaction capabilities for occluded and small
objects in complex scenes. For example, Zhang et al. [10] designed a
cross-modality interactive attention network (CIAN) for multispectral
pedestrian detection, but some occluded pedestrians are missed during
the detection processing. Although transformer-based approaches can
model global context through self-attention mechanisms, their high
computational complexity imposes a huge burden on multimodal
information processing. Speciﬁcally, Fang et al. [12] ﬁrstly introduced

2. Related work
2.1. Visible-infrared object detection
Visible (VIS) object detection performance tends to degrade sharply
in low light, adverse weather conditions, and occlusion scenarios. While
infrared (IR) object detection can penetrate smoke and operate in all
weather conditions, it suﬀers from limitations such as texture detail loss,
low resolution, and susceptibility to thermal cross-interference. VIS-IR
object detection technology [6,7,18], by collaboratively utilizing the
rich spatial texture details of the VIS modality and the reliable thermal radiation characteristics of the IR modality, achieves information
complementarity, signiﬁcantly improving the robustness and accuracy
of object detection in complex environments.
For instance, Zhang et al. [19] designed a multimodal object detection model, namely TINet, where an illumination-guided feature weighting module generates feature weights to adaptively fuse inter- and
intra-modality information from VIS and IR modalities, respectively. A
cross-modal aerial remote sensing image object detection network (CRSIOD) was designed in [20] for oriented object classiﬁcation and localization in some challenging environments, such as low-illumination
and strong occlusion, in which an illumination perception module and
an uncertainty-aware module are devised for estimating feature weights
to promote multimodal feature fusion. To address the position shift issue between diﬀerent modalities, an aligned region CNN (AR-CNN) was
introduced in [11] to achieve local cross-modal feature alignment by a
region feature alignment module and region of interest jitter.

caused by modality bias to some extent [13,17]. To improve the detection accuracy and localization performance of objects with small size,

(1) A novel lightweight modal-guided cross-attention fusion network

(2) To capture complementary information and mitigate intermodal
diﬀerences, we design a visible-guided cross-attention block (VG-CAB)
and an infrared-guided cross-attention block (IG-CAB) to respectively
use contextual information from one modality to guide valuable feature
aggregation in another modality and execute adequate information interaction and complementary exploitation between modalities.
(3) To fully integrate multimodal features, a gated fusion block (GFB)
is constructed to estimate feature weights for adaptive feature fusion.

2.2. Lightweight architecture
Lightweight object detection networks have great promise and potential in applications with limited storage resources, such as embedded and mobile devices [21]. Currently, most monomodality-based object detection approaches were proposed. For example, Xue et al. [22]
devised an eﬃcient and lightweight onboard applications object detector, called EL-YOLO, in which a sparsely connected asymptotic feature
pyramid network is proposed to reduce the interference of diﬀerent
layers during feature aggregation to retain great performance under a
lightweight architecture. Dong et al. [23] developed an eﬃcient and
lightweight optimized network (EL-Net) for object detection, where a
feature perception enhancement module is used to capture relevant features to promote the model performance, and a structured pruning algorithm is exploited for compressing the network structure. A lightweight
small object detection algorithm was introduced in [24] to solve missed
and false detections, where a lightweight cross-layer output reconstruction module improves the fusion eﬀect of shallow and deep features to
boost small object detection.
Compared with single-modal lightweight detection models, multimodal lightweight detection approaches are quite rare. Visible-infrared
object detection requires a dual backbone network to extract multispectral features and an additional multispectral feature fusion module to
merge and mine intermodal information, leading to high network complexity, which makes the lightweight design of multimodal object detection models more challenging. For instance, Cao et al. [25] presented
a lightweight transformer for multimodal object detection, in which a
fusion operator based on the PoolFormer [26], namely PoolFuser, is employed for integrating multimodal features extracted from a lightweight
dual backbone network by using a weighted element-wise summation.
A lightweight attention-guided self-modulation feature fusion network
was devised in [27] for multimodal object detection, where a multimodal feature extractor is lightweight, and an attention-guided selfmodulation feature fusion module is proposed to obtain a robust fused

2.3. Cross-attention mechanism
Cross-attention mechanism guides the feature from one modality
to focus and aggregate key information related to another modality,
achieving adaptive feature interaction and complementarity between
two modalities. Hu et al. [17] developed a deformable cross-attention
transformer for VIS-IR pedestrian detection, where a two-stream crossattention mechanism is adopted to extract intermodal inherent spatial
and geometric relationships. Yuan et al. [13] designed a calibrated and
complementary transformer (C2 Former) to solve modality misalignment
and fusion imprecision issues for VIS-IR object detection, in which an
intermodality cross-attention is used to align multimodal features and
capture complementary information, and a modality normalization is
presented to remove modality discrepancy. Later, a cascade alignmentguided transformer was presented in [28] for obtaining fused feature
between VIS and IR modalities, where a translation scale-rotation alignment (TSRA) module is applied for aligning VIS-IR features by correcting
the position, size, and angle deviations of multimodal objects. Shen et al.
[29] designed dual cross-attention transformers to fuse VIS and IR features for multispectral object detection, where iterative cross-attention
is employed to mine complementary information and model global
dependency between modalities. Hu et al. [14] developed an edgeguided illumination-aware interactive learning-based detector (EI2 Det)
to achieve excellent detection performance, where the edge and illumination information of the VIS modality are used as prior knowledge to
improve the localization eﬀect, and a cross-modal interaction module
is constructed by the multi-head cross-attention mechanism to enhance
inter-modality and intra-modality feature representations.
Although the above multispectral detection models have achieved
superior detection performance, their model parameter and computational cost are too high, making them unsuitable for practical applications. Moreover, in the cross-attention mechanisms of the above
detection models, the query and key tokens are taken from diﬀerent
modalities. They may generate inconsistent attention guidance, resulting in unstable cross-modal interaction due to the diﬀerent noise patterns and spatial features of each modality. Our designed visible-guided
cross-attention and infrared-guided cross-attention blocks are not only
lightweight and eﬀective, but also accurately extract complementary
features by using contextual content from one modality to guide the
information aggregation from another modality.

3.3. Modal-guided cross-attention block
Previous excellent detection models, such as the C2 Former [13],
EI2 Det [14], and ICAFusion [29], employ the cross-attention mechanism to interact with multimodal features and extract complementary
information between modalities, where the query and key tokens from
diﬀerent modalities are used to calculate attention weights, and the
aggregated information is mined from the value token based on the
importance of these weights. However, due to semantic conﬂicts between modalities, the generated attention map may contain interfering
information, leading to insuﬃcient and inaccurate aggregation for intermodal information, which in turn aﬀects the model’s detection performance. To address this, we design a modal-guided cross-attention block
to capture complementary information by feature interaction across
modalities, which is composed of a visible-guided cross-attention block
(VG-CAB) and an infrared-guided cross-attention block (IG-CAB), as
shown in Fig. 2(d) and (e), respectively.
Speciﬁcally, the visible-guided cross-attention block (VG-CAB) and
infrared-guided cross-attention block (IG-CAB) aim to extract complementary information by intermodal feature interaction. The VG-CAB utilizes the relationships between VIS features to fuse IR information, thus
injecting complementary information into the VIS feature. The IG-CAB
generates the attention weights between IR features to guide the integration of the VIS feature. Taking the VG-CAB as an example, to reduce
network complexity, we employ three same lightweight projectors to

terms of mAP50, but outperforms it for the comprehensive mAP metric. Further, the EI2 Det has a larger parameter size than the LCAFNet.
These demonstrate that compared to the EI2 Det, our LCAFNet achieves
a more excellent balance between model complexity and performance.
Further, compared with lightweight detection methods such as the PoolFuser [25] and LASFNet [27], our LCAFNet is inferior to the LASFNet in

4.1. Datasets
4.1.1. DroneVehicle
The DroneVehicle dataset [6] is a well-aligned public benchmark,
which is captured by an unmanned aerial vehicle (UAV) in day and night
conditions. This dataset contains 17,990 VIS-IR image pairs for training
and 1469 VIS-IR image pairs for testing, and the size of these images
is 640 × 512 pixels. Since the labels in this dataset are rotated bounding
boxes, we need to perform a rotated multispectral object detection task.
The dataset includes ﬁve categories, namely “van”, “car”, “bus”, “truck”,
and “freight car”.
4.1.2. FLIR
The FLIR dataset [32] is a widely used benchmark, which is collected
in traﬃc scenes during daytime and nighttime. This dataset includes
5142 VIS-IR image pairs with a resolution of 640 × 512, where 4129 and
1013 image pairs are used for model training and testing, respectively,
whose categories contain “bicycle”, “car”, and “person”.
4.1.3. M3 FD
The M3 FD dataset [33] is a commonly used multispectral benchmark, which is taken in complex environments such as low light, smoke,
etc. This dataset owns 4200 VIS-IR image pairs with a resolution of
1024 × 768, where 3360 and 840 VIS-IR image pairs are applied for network training and testing, respectively. It has six categories, namely
“People”, “Truck”, “Lamp”, “Bus”, “Car”, and “Motorcycle”.
4.1.4. MFAD
The MFAD dataset [14] is a well-aligned multispectral benchmark,
which is ﬁlmed in multiple scenes, including campus, road, tunnel,
viaduct, etc. This dataset is composed of 12,194 VIS-IR image pairs with
a size of 1280 × 960 pixels, in which 9751 and 2443 VIS-IR image pairs
are utilized for training and testing of the detection model, respectively.
It exists six categories, namely “EbikeRider”, “Cyclist”, “Car”, “Truck”,
“Bus”, and “Pedestrian”.
4.1.5. LLVIP
The LLVIP dataset [34] is a well-aligned public multispectral benchmark, which is collected in traﬃc scenes from a surveillance perspective. This dataset consists of 15,488 VIS-IR image pairs with a resolution
of 1280 × 1024, in which 12,025 and 3463 VIS-IR image pairs are employed for network training and testing, respectively. It only contains
one “person” category.
4.2. Experimental settings
To optimize network parameters of our detection model, we use
the SGD optimizer with a learning rate 1.0 × 10−2 and a momentum of
0.937, and the weight decay is equal to 5.0 × 10−4 . All experiments are
conducted on the PyTorch framework with two NVIDIA GeForce RTX
3090 GPUs. In the training stage, we adjust the VIS and IR images to
640 × 640 pixels for the DroneVehicle, FLIR, M3 FD, and MFAD datasets,
and 1024 × 1024 pixels for the LLVIP dataset. For all public datasets, data
augmentations such as random rotation and mosaic are applied. Our detection model is trained in 150, 150, 200, and 150 epochs with a batch
size of 32 on the DroneVehicle, FLIR, M3 FD, and MFAD datasets, respectively. For the LLVIP dataset, our model is trained for 100 epochs with
a batch size of 12.

5. Discussion
To fully understand the proposed LCAFNet, we reveal two detection failure cases in Fig. 8, where two pairs of VIS-IR images are derived from the FLIR dataset. For the ﬁrst pair of VIS-IR images, the
missed “person” object is small and located in an area of huge light
change. In the second pair of VIS-IR images, the missing “bicycle” category is located on a street with street lamps at night and is densely obscured. These superimposed interference conditions lead to our LCAFNet
model failing in detection. Our LCAFNet, as a lightweight multimodal
detection model, employs a lightweight backbone network to extract
multi-scale features, resulting in insuﬃcient multispectral feature representation capabilities. Furthermore, our used YOLO detection head
simultaneously predicts bounding boxes, object conﬁdence, and classiﬁcation probabilities. These three tasks have diﬀerent levels of diﬃculty and require diﬀerent features, which may cause conﬂicts and interference between tasks, leading to detection failures. In the future, we
will investigate methods for extracting more discriminative features and

our VG-CAB and IG-CAB for intermodal interaction between queries and
keys obtained from the same modality. Speciﬁcally, for the PCCAB, the
query (𝑄𝑣𝑖𝑠 ) token from VIS modality and the key (𝐾𝑖𝑟 ) and value (𝑉𝑖𝑟 )
tokens from IR modality are fed into a multi-head scaled dot-product
cross-attention (MSDCA), referring to Eq. (4), to perform information
exchange across modalities. Also, the query (𝑄𝑖𝑟 ) token from IR modal9

6. Conclusion
In this paper, we devise a lightweight modal-guided cross-attention
fusion network for visible-infrared object detection. To suﬃciently exploit complementary features and strengthen feature representation,
we develop a lightweight parallel cross-attention block, containing a
